## Main Features

基于兴趣的社交，功能可参考微博

* 用户
  * 注册登录
    * 使用手机号码登录
    * 需要填写生日，未满14岁禁止注册
  * 管理员账户特殊权限
    * 封禁/解禁用户
    * 删帖子
    * 删评论
* 动态
  * 发布
  * 转发
  * 查看
  * 点赞
  * 收藏
  * 修改
  * 删除
  * 评论
* 各种计数器
  * 评论数
  * 转发数
  * 点赞数
  * 收藏数
* 推荐算法
  * 最近常浏览
