# Project Introduction

> ⚠️  A document demo of how to organize your project introduction

## Main Features

基于兴趣的社交，功能可参考微博

* 用户
  * 注册登录
    * 使用手机号码登录
    * 需要填写生日，未满14岁禁止注册
  * 管理员账户特殊权限
    * 封禁/解禁用户
    * 删帖子
    * 删评论
* 动态
  * 发布
    * 支持附带图片，视频（.mp4格式）
    * 附加可见范围-全部/好友/自己
  * 转发
    * 附带评论
  * 查看
    * 点击图片查看放大图片
  * 点赞
    * 点赞完成图标变化
  * 收藏
    * 收藏完成图标变化
  * 修改
  * 删除
  * 评论
* 各种计数器
  * 评论数
  * 转发数
  * 点赞数
  * 收藏数
* 推荐算法
  * 最近常浏览
